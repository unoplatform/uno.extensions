# Golden Files — Kiota CLI Reference Output

These files were generated by the **Kiota CLI v1.22.3** and serve as the
reference output for parity tests.  The source generator must produce C#
code that matches these files (modulo acceptable differences such as version
strings and whitespace).

## How to Regenerate

If the pinned Kiota runtime version changes, regenerate the golden files:

```powershell
# Ensure the same Kiota CLI version is installed
dotnet tool update --global Microsoft.OpenApi.Kiota --version 1.22.3

$testData = "src/Uno.Extensions.Http.Kiota.Generator.Tests/TestData"
$golden   = "src/Uno.Extensions.Http.Kiota.Generator.Tests/GoldenFiles"

$specs = @(
    @{ Spec = "petstore.json";        Class = "PetStoreClient";        NS = "PetStore.Client"        },
    @{ Spec = "inheritance.json";     Class = "InheritanceClient";     NS = "Inheritance.Client"     },
    @{ Spec = "composed-types.json";  Class = "ComposedTypesClient";   NS = "ComposedTypes.Client"   },
    @{ Spec = "enums.json";           Class = "EnumsClient";           NS = "Enums.Client"           },
    @{ Spec = "error-responses.json"; Class = "ErrorResponsesClient";  NS = "ErrorResponses.Client"  }
)

foreach ($s in $specs) {
    $name = [System.IO.Path]::GetFileNameWithoutExtension($s.Spec)
    kiota generate `
        --openapi  "$testData/$($s.Spec)" `
        --output   "$golden/$name" `
        --language CSharp `
        --class-name   $s.Class `
        --namespace-name $s.NS `
        --log-level Warning `
        --clean-output `
        --exclude-backward-compatible
    # Remove non-code metadata
    Remove-Item "$golden/$name/kiota-lock.json" -ErrorAction SilentlyContinue
    Remove-Item "$golden/$name/.kiota.log"      -ErrorAction SilentlyContinue
}
```

## Specs and Their Purposes

| Spec | Golden Dir | Tests |
|------|-----------|-------|
| `petstore.json` | `petstore/` | Basic CRUD, models, query parameters, nested paths |
| `inheritance.json` | `inheritance/` | `allOf` inheritance, discriminators, base/derived classes |
| `composed-types.json` | `composed-types/` | `oneOf`/`anyOf` union/intersection types |
| `enums.json` | `enums/` | Enum generation, `[Flags]`, `[EnumMember]` |
| `error-responses.json` | `error-responses/` | Error models implementing `IApiErrorClass` |

## Generation Options

All specs use:
- `--exclude-backward-compatible` — no obsolete/compat shims
- `--additional-data true` (default) — models include `AdditionalData`
- Default serializers/deserializers (JSON, Text, Form, Multipart)
