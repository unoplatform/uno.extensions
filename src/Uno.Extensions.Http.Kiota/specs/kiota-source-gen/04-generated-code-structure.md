# Generated Code Structure

## Summary

This document specifies the exact C# code patterns that the Kiota code generator produces. Both Phase 1 (MSBuild Task) and Phase 2 (Source Generator) must produce output conforming to these patterns to maintain compatibility with the Kiota runtime libraries and the existing `AddKiotaClient<T>()` DI registration in `Uno.Extensions.Http.Kiota`.

## Reference: Existing Generated Code

The test harness at `testing/TestHarness/TestHarness/Ext/Http/Client/` contains Kiota v1.16.0-generated code that serves as the canonical reference.

---

## 1. Client Root Class

The root client class is the entry point to the generated API client. It extends `BaseRequestBuilder` and registers the default serializer/deserializer factories.

### Pattern

```csharp
// <auto-generated/>
#pragma warning disable CS0618
using Microsoft.Kiota.Abstractions.Extensions;
using Microsoft.Kiota.Abstractions.Serialization;
using Microsoft.Kiota.Abstractions;
using Microsoft.Kiota.Serialization.Form;
using Microsoft.Kiota.Serialization.Json;
using Microsoft.Kiota.Serialization.Multipart;
using Microsoft.Kiota.Serialization.Text;
using System.Collections.Generic;
using System.IO;
using System.Threading.Tasks;
using System;
namespace {RootNamespace}
{
    [global::System.CodeDom.Compiler.GeneratedCode("Kiota", "{Version}")]
    public partial class {ClientName} : BaseRequestBuilder
    {
        // Navigation properties (see Section 2)

        /// <summary>
        /// Instantiates a new <see cref="global::{RootNamespace}.{ClientName}"/>
        /// and sets the default values.
        /// </summary>
        /// <param name="requestAdapter">The request adapter to use to execute the requests.</param>
        public {ClientName}(IRequestAdapter requestAdapter)
            : base(requestAdapter, "{urlTemplate}", new Dictionary<string, object>())
        {
            ApiClientBuilder.RegisterDefaultSerializer<JsonSerializationWriterFactory>();
            ApiClientBuilder.RegisterDefaultSerializer<TextSerializationWriterFactory>();
            ApiClientBuilder.RegisterDefaultSerializer<FormSerializationWriterFactory>();
            ApiClientBuilder.RegisterDefaultSerializer<MultipartSerializationWriterFactory>();
            ApiClientBuilder.RegisterDefaultDeserializer<JsonParseNodeFactory>();
            ApiClientBuilder.RegisterDefaultDeserializer<TextParseNodeFactory>();
            ApiClientBuilder.RegisterDefaultDeserializer<FormParseNodeFactory>();
            if (string.IsNullOrEmpty(requestAdapter.BaseUrl))
            {
                requestAdapter.BaseUrl = "{baseUrl from spec}";
            }
            PathParameters.TryAdd("baseurl", requestAdapter.BaseUrl);
        }
    }
}
```

### DI Compatibility

`AddKiotaClient<T>()` in `ServiceCollectionExtensions.cs` creates clients via:

```csharp
Activator.CreateInstance(typeof(TClient), requestAdapter)
```

The root client's constructor must have exactly the signature `(IRequestAdapter requestAdapter)` for this to work.

## 2. Request Builders

Request builders represent API path segments and provide navigation + HTTP operation methods.

### Pattern

```csharp
// <auto-generated/>
#pragma warning disable CS0618
using Microsoft.Kiota.Abstractions.Extensions;
using Microsoft.Kiota.Abstractions.Serialization;
using Microsoft.Kiota.Abstractions;
using System.Collections.Generic;
using System.IO;
using System.Threading.Tasks;
using System.Threading;
using System;
namespace {Namespace}
{
    [global::System.CodeDom.Compiler.GeneratedCode("Kiota", "{Version}")]
    public partial class {Name}RequestBuilder : BaseRequestBuilder
    {
        // === Navigation Properties ===
        // One per child path segment (non-parameterized)
        public global::{ChildNamespace}.{ChildName}RequestBuilder {ChildName}
        {
            get => new global::{ChildNamespace}.{ChildName}RequestBuilder(
                PathParameters, RequestAdapter);
        }

        // === Indexer (parameterized segments) ===
        // For path segments like /users/{user-id}
        public global::{ChildNamespace}.{ItemName}RequestBuilder this[string position]
        {
            get
            {
                var urlTplParams = new Dictionary<string, object>(PathParameters);
                urlTplParams.Add("{paramName}", position);
                return new global::{ChildNamespace}.{ItemName}RequestBuilder(
                    urlTplParams, RequestAdapter);
            }
        }

        // === Constructors ===
        public {Name}RequestBuilder(
            Dictionary<string, object> pathParameters,
            IRequestAdapter requestAdapter)
            : base(requestAdapter, "{urlTemplate}", pathParameters)
        {
        }

        public {Name}RequestBuilder(
            string rawUrl,
            IRequestAdapter requestAdapter)
            : base(requestAdapter, "{urlTemplate}", rawUrl)
        {
        }

        // === HTTP Operation Methods ===
        // One per HTTP method on this path

        // === Request Information Builders ===
        // One per HTTP method

        // === WithUrl Method ===
        public {Name}RequestBuilder WithUrl(string rawUrl)
        {
            return new {Name}RequestBuilder(rawUrl, RequestAdapter);
        }

        // === Request Configuration (deprecated in current Kiota) ===
    }
}
```

### HTTP Executor Methods

For each HTTP operation (GET, POST, PUT, PATCH, DELETE):

```csharp
/// <summary>
/// {OperationDescription}
/// </summary>
/// <returns>A <see cref="global::{ReturnType}"/></returns>
/// <param name="cancellationToken">Cancellation token</param>
/// <exception cref="global::{ErrorType4XX}">When receiving 4XX status code</exception>
/// <exception cref="global::{ErrorType5XX}">When receiving 5XX status code</exception>
#if NETSTANDARD2_1_OR_GREATER || NETCOREAPP3_1_OR_GREATER
#nullable enable
public async Task<global::{ReturnType}?> {Method}Async(
    {RequestBodyParam}
    Action<RequestConfiguration<global::{QueryParamsType}>>? requestConfiguration = default,
    CancellationToken cancellationToken = default)
{
#nullable restore
#else
public async Task<global::{ReturnType}> {Method}Async(
    {RequestBodyParam}
    Action<RequestConfiguration<global::{QueryParamsType}>> requestConfiguration = default,
    CancellationToken cancellationToken = default)
{
#endif
    var requestInfo = To{Method}RequestInformation({bodyArg}requestConfiguration);
    var errorMapping = new Dictionary<string, ParsableFactory<IParsable>>
    {
        { "4XX", global::{ErrorType4XX}.CreateFromDiscriminatorValue },
        { "5XX", global::{ErrorType5XX}.CreateFromDiscriminatorValue },
    };
    return await RequestAdapter.SendAsync<global::{ReturnType}>(
        requestInfo, global::{ReturnType}.CreateFromDiscriminatorValue,
        errorMapping, cancellationToken).ConfigureAwait(false);
}
```

### Request Information Builders

```csharp
public RequestInformation To{Method}RequestInformation(
    {RequestBodyParam}
    Action<RequestConfiguration<global::{QueryParamsType}>> requestConfiguration = default)
{
    var requestInfo = new RequestInformation(Method.{METHOD}, UrlTemplate, PathParameters);
    requestInfo.Configure(requestConfiguration);
    requestInfo.Headers.TryAdd("Accept", "{acceptHeader}");
    // If request body:
    // requestInfo.SetContentFromParsable(RequestAdapter, "{contentType}", body);
    return requestInfo;
}
```

### Query Parameter Classes

For operations with query parameters:

```csharp
[global::System.CodeDom.Compiler.GeneratedCode("Kiota", "{Version}")]
public partial class {MethodName}QueryParameters
{
    [QueryParameter("{wireFormatName}")]
    public int? Top { get; set; }

    [QueryParameter("$filter")]
    public string Filter { get; set; }

    [QueryParameter("$select")]
    public string[] Select { get; set; }
}
```

### Deprecated Request Configuration (Backward Compatibility)

When `ExcludeBackwardCompatible` is `false`:

```csharp
[Obsolete("This class is deprecated. Please use the generic RequestConfiguration class.")]
[global::System.CodeDom.Compiler.GeneratedCode("Kiota", "{Version}")]
public partial class {MethodName}RequestConfiguration
    : RequestConfiguration<global::{QueryParamsType}>
{
}
```

## 3. Model Classes (IParsable)

Models represent request/response body schemas.

### Basic Pattern

```csharp
// <auto-generated/>
#pragma warning disable CS0618
using Microsoft.Kiota.Abstractions.Extensions;
using Microsoft.Kiota.Abstractions.Serialization;
using System.Collections.Generic;
using System.IO;
using System;
namespace {Namespace}.Models
{
    [global::System.CodeDom.Compiler.GeneratedCode("Kiota", "{Version}")]
    public partial class {ModelName} : IParsable
    {
        // === Properties ===
#if NETSTANDARD2_1_OR_GREATER || NETCOREAPP3_1_OR_GREATER
#nullable enable
        public string? PropertyName { get; set; }
#nullable restore
#else
        public string PropertyName { get; set; }
#endif

        // Non-nullable value types don't need the conditional:
        public int Count { get; set; }

        // Collection properties:
#if NETSTANDARD2_1_OR_GREATER || NETCOREAPP3_1_OR_GREATER
#nullable enable
        public List<global::{Namespace}.Models.{ItemType}>? Items { get; set; }
#nullable restore
#else
        public List<global::{Namespace}.Models.{ItemType}> Items { get; set; }
#endif

        // === Factory Method ===
        public static global::{Namespace}.Models.{ModelName}
            CreateFromDiscriminatorValue(IParseNode parseNode)
        {
            _ = parseNode ?? throw new ArgumentNullException(nameof(parseNode));
            return new global::{Namespace}.Models.{ModelName}();
        }

        // === Deserialization ===
        public virtual IDictionary<string, Action<IParseNode>>
            GetFieldDeserializers()
        {
            return new Dictionary<string, Action<IParseNode>>
            {
                { "propertyName", n => { PropertyName = n.GetStringValue(); } },
                { "count", n => { Count = n.GetIntValue() ?? 0; } },
                { "items", n => {
                    Items = n.GetCollectionOfObjectValues<
                        global::{Namespace}.Models.{ItemType}>(
                        global::{Namespace}.Models.{ItemType}
                            .CreateFromDiscriminatorValue)?.AsList();
                } },
            };
        }

        // === Serialization ===
        public virtual void Serialize(ISerializationWriter writer)
        {
            _ = writer ?? throw new ArgumentNullException(nameof(writer));
            writer.WriteStringValue("propertyName", PropertyName);
            writer.WriteIntValue("count", Count);
            writer.WriteCollectionOfObjectValues<
                global::{Namespace}.Models.{ItemType}>("items", Items);
        }
    }
}
```

### Deserialization Method Dispatch Table

The `GetFieldDeserializers` method maps JSON property names to parsing lambdas. The specific `IParseNode` method depends on the property type:

| C# Type | IParseNode Method |
|---------|-------------------|
| `string` | `GetStringValue()` |
| `bool` / `bool?` | `GetBoolValue()` |
| `int` / `int?` | `GetIntValue()` |
| `long` / `long?` | `GetLongValue()` |
| `float` / `float?` | `GetFloatValue()` |
| `double` / `double?` | `GetDoubleValue()` |
| `decimal` / `decimal?` | `GetDecimalValue()` |
| `Guid` / `Guid?` | `GetGuidValue()` |
| `DateTimeOffset` / `DateTimeOffset?` | `GetDateTimeOffsetValue()` |
| `DateOnly` / `DateOnly?` | `GetDateValue()` |
| `TimeOnly` / `TimeOnly?` | `GetTimeValue()` |
| `TimeSpan` / `TimeSpan?` | `GetTimeSpanValue()` |
| `byte[]` | `GetByteArrayValue()` |
| `Stream` | `GetByteArrayValue()` → wrap in `MemoryStream` |
| `T` (object) | `GetObjectValue<T>(T.CreateFromDiscriminatorValue)` |
| `List<string>` | `GetCollectionOfPrimitiveValues<string>()?.AsList()` |
| `List<T>` (objects) | `GetCollectionOfObjectValues<T>(T.CreateFromDiscriminatorValue)?.AsList()` |
| `T` (enum) | `GetEnumValue<T>()` |
| `List<T>` (enums) | `GetCollectionOfEnumValues<T>()?.AsList()` |

### Serialization Method Dispatch Table

The `Serialize` method writes each property. The specific `ISerializationWriter` method depends on the property type:

| C# Type | ISerializationWriter Method |
|---------|----------------------------|
| `string` | `WriteStringValue()` |
| `bool` / `bool?` | `WriteBoolValue()` |
| `int` / `int?` | `WriteIntValue()` |
| `long` / `long?` | `WriteLongValue()` |
| `float` / `float?` | `WriteFloatValue()` |
| `double` / `double?` | `WriteDoubleValue()` |
| `decimal` / `decimal?` | `WriteDecimalValue()` |
| `Guid` / `Guid?` | `WriteGuidValue()` |
| `DateTimeOffset` / `DateTimeOffset?` | `WriteDateTimeOffsetValue()` |
| `DateOnly` / `DateOnly?` | `WriteDateValue()` |
| `TimeOnly` / `TimeOnly?` | `WriteTimeValue()` |
| `TimeSpan` / `TimeSpan?` | `WriteTimeSpanValue()` |
| `byte[]` | `WriteByteArrayValue()` |
| `T` (object) | `WriteObjectValue<T>()` |
| `List<string>` | `WriteCollectionOfPrimitiveValues<string>()` |
| `List<T>` (objects) | `WriteCollectionOfObjectValues<T>()` |
| `T` (enum) | `WriteEnumValue<T>()` |
| `List<T>` (enums) | `WriteCollectionOfEnumValues<T>()` |

### Additional Data Properties

When `IncludeAdditionalData` is `true`:

```csharp
public partial class {ModelName} : IAdditionalDataHolder, IParsable
{
    public IDictionary<string, object> AdditionalData { get; set; }

    // In constructor or initializer:
    // AdditionalData = new Dictionary<string, object>();

    // In GetFieldDeserializers: no entry needed (handled by runtime)

    // In Serialize:
    // writer.WriteAdditionalData(AdditionalData);
}
```

### Backing Store Properties

When `UsesBackingStore` is `true`:

```csharp
public partial class {ModelName} : IBackedModel, IParsable
{
    public IBackingStore BackingStore { get; private set; }

    public {ModelName}()
    {
        BackingStore = BackingStoreFactorySingleton.Instance.CreateBackingStore();
        AdditionalData = new Dictionary<string, object>();
    }

    // Property pattern with backing store:
    public string PropertyName
    {
        get { return BackingStore?.Get<string>("propertyName"); }
        set { BackingStore?.Set("propertyName", value); }
    }
}
```

## 4. Models with Inheritance (allOf)

When a schema uses `allOf` with a `$ref`:

```csharp
// Base class
public partial class Animal : IParsable
{
    public string Name { get; set; }
    public string Type { get; set; }

    // Standard IParsable members...
}

// Derived class
public partial class Cat : global::{Namespace}.Models.Animal, IParsable
{
    public string Color { get; set; }

    // Overrides base factory
    public new static global::{Namespace}.Models.Cat
        CreateFromDiscriminatorValue(IParseNode parseNode)
    {
        _ = parseNode ?? throw new ArgumentNullException(nameof(parseNode));
        return new global::{Namespace}.Models.Cat();
    }

    // Merges base + derived fields
    public override IDictionary<string, Action<IParseNode>>
        GetFieldDeserializers()
    {
        return new Dictionary<string, Action<IParseNode>>(
            base.GetFieldDeserializers())
        {
            { "color", n => { Color = n.GetStringValue(); } },
        };
    }

    // Serializes base + derived
    public override void Serialize(ISerializationWriter writer)
    {
        _ = writer ?? throw new ArgumentNullException(nameof(writer));
        base.Serialize(writer);
        writer.WriteStringValue("color", Color);
    }
}
```

### Discriminated Inheritance

When `discriminator` is present:

```csharp
public static global::{Namespace}.Models.Animal
    CreateFromDiscriminatorValue(IParseNode parseNode)
{
    _ = parseNode ?? throw new ArgumentNullException(nameof(parseNode));
    var mappingValue = parseNode.GetChildNode("{discriminatorProperty}")
        ?.GetStringValue();
    return mappingValue switch
    {
        "#/components/schemas/Cat" =>
            new global::{Namespace}.Models.Cat(),
        "#/components/schemas/Dog" =>
            new global::{Namespace}.Models.Dog(),
        _ => new global::{Namespace}.Models.Animal(),
    };
}
```

## 5. Composed Types (oneOf / anyOf)

### Union Types (oneOf)

```csharp
[global::System.CodeDom.Compiler.GeneratedCode("Kiota", "{Version}")]
public partial class {TypeName} : IComposedTypeWrapper, IParsable
{
    // One property per composed member
    public global::{Namespace}.Models.TypeA? TypeAMember { get; set; }
    public global::{Namespace}.Models.TypeB? TypeBMember { get; set; }
    public string? StringMember { get; set; }

    public static global::{Namespace}.Models.{TypeName}
        CreateFromDiscriminatorValue(IParseNode parseNode)
    {
        _ = parseNode ?? throw new ArgumentNullException(nameof(parseNode));
        var result = new global::{Namespace}.Models.{TypeName}();
        // Try each type in order
        if (parseNode.GetChildNode("{discriminator}")?.GetStringValue() is string disc)
        {
            // Use discriminator if available
        }
        return result;
    }

    public IDictionary<string, Action<IParseNode>> GetFieldDeserializers()
    {
        // Delegate to the active composed member
        if (TypeAMember != null)
            return TypeAMember.GetFieldDeserializers();
        if (TypeBMember != null)
            return TypeBMember.GetFieldDeserializers();
        return new Dictionary<string, Action<IParseNode>>();
    }

    public void Serialize(ISerializationWriter writer)
    {
        _ = writer ?? throw new ArgumentNullException(nameof(writer));
        if (TypeAMember != null)
            writer.WriteObjectValue<global::{Namespace}.Models.TypeA>(null, TypeAMember);
        else if (TypeBMember != null)
            writer.WriteObjectValue<global::{Namespace}.Models.TypeB>(null, TypeBMember);
        else if (StringMember != null)
            writer.WriteStringValue(null, StringMember);
    }
}
```

### Intersection Types (anyOf)

Similar to union types, but multiple members may be set simultaneously:

```csharp
public void Serialize(ISerializationWriter writer)
{
    _ = writer ?? throw new ArgumentNullException(nameof(writer));
    // All non-null members are serialized
    if (TypeAMember != null)
        writer.WriteObjectValue<global::{Namespace}.Models.TypeA>(null, TypeAMember);
    if (TypeBMember != null)
        writer.WriteObjectValue<global::{Namespace}.Models.TypeB>(null, TypeBMember);
}
```

## 8. URL Template Format

Kiota uses RFC 6570 URI templates:

```
{+baseurl}/users{?%24top,%24skip,%24filter,%24select,%24orderby,%24count}
```

Components:
- `{+baseurl}` — reserved expansion (no percent-encoding of the base URL)
- Literal path segments (`/users`)
- `{paramName}` — simple string expansion for path parameters
- `{?param1,param2}` — form-style query expansion
- `%24` — escaped `$` for OData query parameters

## 9. File Naming Conventions

Kiota follows these naming conventions:

| CodeDOM Element | File Name Pattern | Example |
|----------------|-------------------|---------|
| Root Client | `{ClientName}.cs` | `PetStoreClient.cs` |
| Request Builder | `{SegmentName}RequestBuilder.cs` | `UsersRequestBuilder.cs` |
| Item Request Builder | `{SegmentName}ItemRequestBuilder.cs` | `UsersItemRequestBuilder.cs` |
| Model | `{SchemaName}.cs` | `User.cs` |
| Error Model | `{SchemaName}.cs` | `ODataError.cs` |
| Enum | `{EnumName}.cs` | `UserStatus.cs` |

### Directory Structure

```
Generated/
├── {ClientName}.cs
├── {Segment1}/
│   ├── {Segment1}RequestBuilder.cs
│   ├── {Segment2}/
│   │   ├── {Segment2}RequestBuilder.cs
│   │   └── Item/
│   │       └── {Segment2}ItemRequestBuilder.cs
│   └── ...
└── Models/
    ├── {Model1}.cs
    ├── {Model2}.cs
    ├── {Enum1}.cs
    └── ...
```

For the source generator (Phase 2), hint names follow the pattern:
```
{ClientName}.{Namespace}.{TypeName}.g.cs
```

## 10. Code Conventions Summary

| Aspect | Convention |
|--------|-----------|
| All type references | `global::` prefixed |
| Generated code attribute | `[global::System.CodeDom.Compiler.GeneratedCode("Kiota", "{version}")]` |
| Nullable reference types | `#if NETSTANDARD2_1_OR_GREATER || NETCOREAPP3_1_OR_GREATER` conditional |
| Access modifier | `public partial class` (configurable via `TypeAccessModifier`) |
| Base class for builders | `BaseRequestBuilder` |
| Interface for models | `IParsable` (+ `IAdditionalDataHolder`, `IBackedModel` optionally) |
| Async method naming | `{Method}Async` |
| Cancellation | `CancellationToken cancellationToken = default` |
| Null guards | `_ = value ?? throw new ArgumentNullException(nameof(value))` |
| Collection initialization | `.AsList()` extension from `Microsoft.Kiota.Abstractions.Extensions` |
| Pragma suppression | `#pragma warning disable CS0618` (for deprecated members) |
| XML doc comments | Present on all public members |
